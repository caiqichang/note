# JavaScript异步相关

## 1. Promise

```javascript
// 定义
new Promise((resolve, reject) => {
    // success 
    resolve();
    // fail
    reject();
});

// 调用
promiseInstance()
    .then(callback)
    .catch(callback)
    .finally(callback); // 无论成功或失败都会执行

// 如果resolve也是Promise或返回Promise，可以采用链式then。
promiseInstance()
    .then(promise)
    .then(promise)
    // ...
    .catch(callback) // 只需要一个catch，任何一个promise执行reject都会调用，后续promise中止
    .finally(callback); 
```

Promise.all() -- 所有Promise都成功才resolve

Promise.allSettled() -- 返回所有Promise结果

Promise.any() -- 所有Promise都失败才reject

Promise.race() -- 返回第一个结束（最快）的Promise的resolve或reject

## 2. async/await
```javascript
// async 后面必须是关键字function
async function asyncFunction() {
    // await 后面必须是返回promise的方法
    await sept1();
    // do something or return 
    await sept2();
    // ...
}
```
