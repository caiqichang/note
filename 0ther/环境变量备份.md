# 环境变量备份

- 配置在 `用户环境变量` 而非 `系统环境变量` 防止环境变量失效
- 修改环境变量后可能需要重启计算机
- PATH中的第一个环境变量(如下)作用未知，但没有的话会出现奇怪的问题

### PATH
```
%USERPROFILE%\AppData\Local\Microsoft\WindowsApps
%JAVA_HOME%\bin
%M2_HOME%\bin
%NODEJS%\nodejs
%NODEJS%\node_global
```

### JAVA_HOME 
D:\openjdk\openjdk
### java11
D:\openjdk11\openjdk11\bin
### M2_HOME  
D:\maven\maven
### MAVEN_USER_HOME
D:\maven
### GRADLE_USER_HOME
D:\gradle\repository
### NODEJS
D:\nodejs
