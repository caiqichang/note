# 注册表 -- 右键菜单

## 文件右键
* 注册表位置 -- HKEY_CLASSES_ROOT/*/shell 
* 新建项 -- 菜单名称
* 新建子项 -- command
* 设置值 -- "执行程序绝对路径" "%1"

## 文件夹右键
* 注册表位置 -- HKEY_CLASSES_ROOT/Directory/shell 
* 新建项 -- 菜单名称
* 新建子项 -- command
* 设置值 -- "执行程序绝对路径" "%1"

## 文件夹内空白处右键
* 注册表位置 -- HKEY_CLASSES_ROOT/Directory/Background/shell 
* 新建项 -- 菜单名称
* 新建子项 -- command
* 设置值 -- "执行程序绝对路径" "%v."
