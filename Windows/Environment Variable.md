# Environment Variable 

> Setting at `User` variable instead of `System`.

## Path 
> Attention: do not remove the first variable!
```
%USERPROFILE%\AppData\Local\Microsoft\WindowsApps
%JAVA_HOME%\bin
%M2_HOME%\bin
%NODEJS%\nodejs
%NODEJS%\node_global
```

| Variable | Value |
| - | - |
| JAVA_HOME | D:\openjdk\openjdk |
| java11 | D:\openjdk11\openjdk11\bin |
| M2_HOME | D:\maven\maven |
| MAVEN_USER_HOME | D:\maven |
| GRADLE_USER_HOME | D:\gradle\repository |
| NODEJS | D:\nodejs |
