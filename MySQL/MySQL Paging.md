# MySQL Paging

- PageNumber is 0-base.

## LIMIT OFFSET
```sql
SELECT b.*, ... 
FROM business AS b LEFT JOIN ... 
WHERE ... 
ORDER BY ... 
LIMIT ${PageSize} OFFSET (${PageNumber} * ${PageSize});
```

## Without OFFSET (Recommend)
```sql
SELECT * FROM (
    SELECT (@row:=@row+1) AS rownum,
        b.*, ...
    FROM (SELECT @row:=-1) AS RowTable,
        business AS b
    LEFT JOIN ...
    WHERE ...
    ORDER BY ...
) AS PageData WHERE PageData.rownum >= (${PageNumber} * ${PageSize}) LIMIT ${PageSize};

```
